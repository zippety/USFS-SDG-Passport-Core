## 2. The Product (The "What")
The App is currently LIVE (Localhost MVP) with the following features:
*   **SDG Passport**: Digital stamp collection for all 17 Goals.
*   **Sustainability Pet**: ("Eco-Companion") An interactive mascot to drive student engagement.
*   **Events Calendar**: (`/events`) A booking system for workshops and volunteering.
*   **Community Hub**: (`/community`) A Slack-like chat interface to replace WhatsApp.
*   **Team Command Center**: (`/team`) Internal dashboard for Exec alignment (Strategy, Roles, Kanban).
